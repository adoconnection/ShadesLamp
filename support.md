# Support

**Shades Lamp**

## Getting Help

If you need help with the Shades Lamp app or your LED lamp device, please use one of the following options:

### Report a Bug or Request a Feature

Open an issue on our GitHub repository:
[github.com/adoconnection/ShadesLamp/issues](https://github.com/adoconnection/ShadesLamp/issues)

### Frequently Asked Questions

**Q: The app cannot find my lamp.**
A: Make sure Bluetooth is enabled on your phone and the lamp is powered on. On Android, location permission is required for BLE scanning — this is an Android OS requirement and the app does not collect location data.

**Q: The connection drops frequently.**
A: Stay within Bluetooth range (approximately 10 meters). Walls and obstacles may reduce the effective range. Try moving closer to the lamp.

**Q: A program is not working correctly.**
A: Try reinstalling the program from the marketplace. If the issue persists, open a GitHub issue with the program name and a description of the problem.

**Q: How do I reset the lamp to default settings?**
A: Power cycle the lamp (turn off and on). The lamp will restart with the last active program.

**Q: How do I create my own LED programs?**
A: LED programs are written in C and compiled to WebAssembly. See the documentation in our GitHub repository for the programming guide and API reference.
